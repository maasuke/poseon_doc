プライバシーポリシー

本プライバシーポリシーは、モバイルアプリ「PoseOn!」（以下「本サービス」）に適用されます。 本サービスは無料で提供され、現状有姿（AS IS）で提供されます。

1. 取得する情報

本サービスは、次の情報を取得します。

  • ユーザーID：本サービス初回利用時にサーバーでランダムに生成される識別子。 本サービスはこのIDを端末内の安全な保存領域に保存し、以後の処理に利用します。
  • 画像：利用者がカメラまたは写真ライブラリから選択した「背景画像」。 背景画像はAIによるポーズ提案画像の生成のために、本サービスのクラウド（AWS）へアップロードされます。
  • 設定情報：人数、雰囲気（ポーズの大きさ/個性）、画角、追加指示など、利用者が本サービスで選択/入力した内容。
  • 生成用プロンプト：上記の設定情報から本サービスが生成するテキスト（AIへの指示文）。
  • サーバーのアクセスログ等：本サービス基盤（AWS）において、セキュリティ確保・障害対応のために自動的に記録される情報 （例：IPアドレス、User-Agent、日時、リクエストID、エラー情報等）。

位置情報について：本サービスは、モバイル端末の正確な位置情報（GPS等）を取得しません。

2. 利用目的

取得した情報は、次の目的で利用します。

  • ポーズ提案画像の生成：背景画像のアップロード、AI処理の実行、生成結果の返却
  • 生成状況の提供：生成処理の進捗・成功/失敗の表示
  • 不正利用防止・セキュリティ確保：アクセスログ等の分析、障害/不具合対応
  • サポート対応：お問い合わせへの対応（必要な場合）

3. AI（人工知能）の利用と送信される情報

本サービスは、背景画像にポーズ提案の人物を合成した画像を生成するためにAI技術を利用します。 本サービスのバックエンドは、OpenAI が提供する画像編集APIを利用します。 この処理のため、本サービスは OpenAI に対して、背景画像および生成用プロンプトを送信します。 送信された情報は、ポーズ提案画像の生成のために処理されます。

4. 第三者提供・委託

本サービスは、機能提供のために以下の外部事業者を利用します。各事業者は、本サービスの指示に基づき必要な範囲で情報を取り扱います。

  • Amazon Web Services（AWS）（運用リージョン：ap-northeast-1）
  • OpenAI：背景画像とプロンプトの処理（画像編集APIによる生成）

本サービスは、法令に基づく場合、または権利保護・安全確保・不正調査等のために必要であると合理的に判断した場合に、取得情報を開示することがあります。

5. 端末内での保存

本サービスは、以下を端末内に保存する場合があります。

  • userId：端末内の安全な保存領域に保存されます。
  • 生成画像の一時ファイル：生成されたポーズ提案画像は、表示のため端末の一時ディレクトリに保存されます（端末OSの仕様により、一定期間後に自動的に削除される場合があります）。
  • 写真ライブラリへの保存：利用者が操作した場合に限り、撮影した写真を端末の写真ライブラリへ保存します。

6. データの保持期間

本サービスは、次の方針でデータを保持します。

  • 背景画像：AWS S3 に保存されます。 アップロードから48時間以内に自動的に削除されます。
  • 生成画像：AWS S3 に保存されます。 作成から48時間以内に自動的に削除されます。
  • ステータス情報：生成の進捗/結果を表すJSONがAWS S3に保存されます。 作成から48時間以内に自動的に削除されます。
  • アクセスログ等：本サービス基盤（AWS CloudWatch 等）におけるログは、 法令遵守、セキュリティ確保および障害対応の目的に必要な期間、保持されます。

OpenAI に送信された情報の管理および利用については、OpenAI, Inc. のデータ利用方針に従い取り扱われます。
詳細は、以下の OpenAI 公式ドキュメントをご参照ください：
https://platform.openai.com/docs/guides/your-data

7. オプトアウト（取得の停止）

利用者は、本サービスをアンインストールすることで、以後の情報取得を停止できます。アンインストールは、端末の標準手順またはアプリストア等の提供する手順をご利用ください。

8. セキュリティ

本サービスは、取得情報を保護するために、合理的な安全対策を講じます。例えば、画像のアップロード/ダウンロードには有効期限付きの署名URLを用いるなどの対策を実施しています。

9. 児童のプライバシー

本サービスは、13歳未満の児童から故意に個人情報を収集したり、13歳未満の児童に向けてマーケティングを行ったりしません。 万一、児童が個人情報を提供した可能性がある場合は、下記窓口までご連絡ください。必要な対応を行います。 なお、国によっては、個人情報の処理に同意できる年齢が16歳以上である場合があり、その場合は保護者等の同意が必要となることがあります。
